# VerdeDesk

**Green receipts made simple for expats in Portugal.**

Issue recibos verdes, track your income, and stay compliant with Portuguese tax law — in plain English. Built for D8 visa holders and expat freelancers.

**Live:** [verdedesk.vercel.app](https://verdedesk.vercel.app)

## Free Tools

- [Portugal Freelancer Tax Calculator](https://verdedesk.vercel.app/tools/tax-calculator) — Estimate IRS, social security, and take-home pay
- [NHR 2.0 Eligibility Checker](https://verdedesk.vercel.app/tools/nhr-checker) — Check if you qualify for the 20% flat tax rate
